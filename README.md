# Mini Web Projects Collection

This repository contains a collection of simple web development projects built using **HTML, CSS, and JavaScript**. These projects were created for learning and practice purposes, focusing on core JavaScript logic and basic DOM manipulation.

## Projects Included

- **Tic Tac Toe**
  - A simple two-player Tic Tac Toe game implemented using HTML, CSS, and JavaScript.

- **Random Text Generator**
  - Generates random text/content on button click.

- **Color Changer**
  - Changes background colors dynamically using JavaScript.

- **Palindrome Checker**
  - Checks whether the entered text or number is a palindrome.

- **Stopwatch**
  - A basic stopwatch with start, stop, and reset functionality.

- **To-Do List**
  - A simple task manager to add and remove daily tasks.

- **Weather Web App**
  - Fetches and displays real-time weather data using a **simple public weather API**.

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Public Weather API (for Weather App)

## Purpose

- Practice JavaScript fundamentals
- Improve DOM manipulation skills
- Build small, functional web applications
- Create beginner-friendly projects for portfolio
